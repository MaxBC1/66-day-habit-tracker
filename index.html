Overarching Themes

  • Consistency beats intensity.
  • Three anchor sessions keep the week alive.
  • Two light days protect recovery and momentum.
  • No zero days: a five-minute minimum still counts.
  • Food should be no-cook, lower-cost and protein-forward.
  • The VLAN project moves through 2 × 25-minute blocks per week.
Phase 1 — Days 1-21: Install

Make the routine easy enough to repeat. Keep training at 6/10, protect calves and shins, start water/protein habits, and create the VLAN project document.

Phase 2 — Days 22-44: Build

Add progress without breaking consistency. Runs become longer, kettlebell sessions get stronger, pull-up volume increases, and the VLAN design takes shape.

Phase 3 — Days 45-66: Automate

Make the routine feel normal. Keep the three anchors alive, protect sleep twice weekly, and make the VLAN proposal presentable.

Weekly Fitness Template
  • Monday: Strength A
  • Tuesday: Pull-up + mobility + VLAN
  • Wednesday: Run
  • Thursday: Light aerobic/mobility + VLAN
  • Friday: Strength B
  • Weekend: optional walk, golf, easy run, mobility, or rest
Strength A
  • Pull-up practice
  • Goblet squat
  • Push-ups
  • One-arm KB row
  • KB Romanian deadlift
  • Plank/dead bug
  • Suitcase carry
Strength B
  • Pull-up practice
  • KB swings
  • Reverse lunge/split squat
  • One-arm KB press
  • Double KB deadlift
  • Hollow hold/dead bug
  • Farmer carry
Nutrition Defaults
  • Water before coffee.
  • Protein + fruit by 9:30am on workout days.
  • Three cheaper lunches per week.
  • Microwave dinners stay; add veg, yoghurt, fruit or protein where needed.
  • Sweets are allowed, but deliberate.
VLAN Project Blocks

2 minutes: define today’s output. 20 minutes: work on one output. 3 minutes: write completed, next action and blocker.
