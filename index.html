Overarching Themes

  • Consistency beats intensity.
  • Five planned movement days keep the week alive: 3 workouts, 2 runs, 2 flexible/light days.
  • Flexible/light days protect recovery, social life and momentum.
  • No zero days: a five-minute minimum still counts.
  • Food should be no-cook, lower-cost and protein-forward.
  • Project Work moves through 2 × 25-minute suggested blocks per week.
Phase 1 — Days 1-21: Install

Make the routine easy enough to repeat. Keep training at 6/10, protect calves and shins, start water/light meal habits, and create the Project Work document.

Phase 2 — Days 22-44: Build

Add progress without breaking consistency. Runs become longer, kettlebell sessions get stronger, pull-up volume increases, and the Project Work design takes shape.

Phase 3 — Days 45-66: Automate

Make the routine feel normal. Keep the workout/run rhythm alive, protect sleep, and make the Project Work output presentable.
